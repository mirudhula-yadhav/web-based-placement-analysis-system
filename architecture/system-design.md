# System Design

## Modules

### Student Module

* Student Registration
* Student Login
* View Placement Opportunities
* Apply for Jobs

### Admin Module

* Secure Login
* Manage Placement Posts
* Manage Companies
* Manage Students

### Company Module

* Publish Job Opportunities
* Define Eligibility Criteria
* Review Applications

## Database

The system uses MySQL to store:

* Student Information
* Company Information
* Placement Posts
* Applications

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* AJAX
* jQuery
