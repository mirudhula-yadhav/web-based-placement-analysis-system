# Setup Guide

## Prerequisites

- XAMPP
- PHP
- MySQL
- Web Browser

## Installation Steps

1. Install XAMPP.
2. Start Apache and MySQL services.
3. Create a database named `placement_system`.
4. Import the `database/schema.sql` file.
5. Place the project folder inside the `htdocs` directory.
6. Open the browser and navigate to:

http://localhost/web-based-placement-analysis-system

## Modules

- Student Registration
- Student Login
- Placement Dashboard
- Company Listings

- PHP
