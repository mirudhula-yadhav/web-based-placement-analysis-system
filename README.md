# Web Based Placement Analysis System

## Overview

The Web Based Placement Analysis System is a web application developed to help students, placement officers, and companies manage placement activities efficiently.

The system provides a centralized platform where students can view placement opportunities, register for recruitment drives, and access company information. Placement officers can manage job postings and student records through an administrative dashboard.

This project was developed as part of my Bachelor of Science in Computer Science at Karpagam Academy of Higher Education.

## Technologies Used

* PHP
* MySQL
* HTML
* CSS
* JavaScript
* AJAX
* jQuery

## Features

### Student Module

* Student Registration
* Login Authentication
* View Placement Opportunities
* Apply for Companies
* View Company Information

### Admin Module

* Secure Admin Login
* Manage Placement Posts
* Manage Categories
* Manage Student Records
* Manage Company Information

### System Features

* User Authentication
* Placement Tracking
* Database Management
* Responsive Web Interface

## Project Objectives

* Reduce manual placement management activities.
* Improve communication between students and placement officers.
* Provide centralized access to placement information.
* Maintain secure and organized placement records.
